关于这款游戏

沉浸于扣人心弦的故事中。主角化身为心中的小男孩，与可爱的泰迪熊同行。在寻求内心平静的旅程中，他们一起破解烧脑谜题、逃过可怕怪物，并躲避危险的陷阱。

穿梭于遍布着漂浮城堡、多岩洞穴、童话森林以及波状平原的神祕世界中。沉浸在梦幻的迷人世界里，领悟坚韧不拔的可贵，并学会放手，体验其苦中带甜的美好。

主要特色

— 深入体验错综复杂的故事、探寻秘密，并解开迷人故事中的谜团；

— 认识令人难以忘怀的角色，如乖巧可爱的小格里芬，以及忠诚的泰迪熊同伴比利；

— 指挥你的可爱泰迪熊朋友来帮助你去往难以到达的地点；

— 聆听梦幻的旋律及美妙的音乐，进入难令人难忘的独特世界；

— 挑战性十足的烧脑谜题与惊心动魄的动作场景相得益彰，为你带来精彩刺激的游戏体验。

系统需求

    • 最低配置:
    • 需要 64 位处理器和操作系统
    • 操作系统: Windows 10, 64 bit
    • 处理器: Intel CPU Core i3
    • 内存: 4 GB RAM
    • 显卡: Nvidia GTX 460
    • DirectX 版本: 11
    • 存储空间: 需要 5 GB 可用空间
    • 推荐配置:
    • 需要 64 位处理器和操作系统
    • 操作系统: Windows 10, 64 bit
    • 处理器: Intel CPU Core i7
    • 内存: 8 GB RAM
    • 显卡: Nvidia GPU GeForce GTX 660
    • DirectX 版本: 11
    • 存储空间: 需要 5 GB 可用空间
