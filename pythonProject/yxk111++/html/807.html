【游戏简介】
《Crysis Warhead》，并非《Crysis》的后续，而是《Crysis》中的一个角色的外传。《Crysis Warhead》的主角是Kykes士官长Psycho，他仅仅在《Crysis》中出现几次，但亦神秘地在结尾部分在航空母舰的飞行甲板上。这次将奉命执行秘密任务，深入岛屿的另一边展开激烈的杀戮，不顾一切取得拥有强大科技实力的北韩将领。

火爆杀戮
从字面意思来解说是一种可能发生爆炸，并產生燃烧的弹头，实际是指游戏中那位性急的主角且暴力的主角”Psycho”，他的疯狂杀戮就是游戏的重点了。玩家将感受到全新的作战风貌，不再像前作”Crysis”般需要思考来完成种种的任务，而是比前作更加突出动作性，这正好表现出主角”Psycho”的火爆性格。

《Crysis Warhead》仍保留”Nanosuit”方正玩家去潜行来恢復体力及伏击敌人。Psycho的”nanosuit”性能较桌卓越，可以加驾驭更高级的战斗交通工具和枪械，增加多点新鲜感。

系统需求

    • 最低配置:
    • 支持的操作系统：Microsoft Windows XP Service Pack 2 或者 Microsoft Vista
    • 处理器:Intel Pentium 4 2.8 GHz (3.2 GHz 为 Vista), Intel Core 2.0 GHz (2.2 GHz 为 Vista), AMD Athlon 2800+ (3200+ 为 Vista) 或更好
    • 内存:1.0 GB (XP) 或 1.5 GB (Vista)
    • 显存：256 MB
    • 显卡:NVIDIA GeForce 6800 GT, ATI Radeon 9800 Pro (Vista 至少需要 Radeon X800 Pro ) 或更好
    • DirectX Version:DirectX 9.0c 或者 10
    • 声卡:任何兼容 DirectX 9.0c 的声卡
    • 硬盘剩余空间：15GB
