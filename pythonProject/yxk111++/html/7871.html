关于这款游戏

在这个销量超过120000份的最畅销系列游戏的下一部作品中，准备好面对物流挑战吧！在《Farm Manager 2021》中，作为农场经理，你需要确保收成、员工满意度、动物健康状况、设备效率和农作物加工处于适当水平。你可以购买新土地或者使用废弃建筑中的材料来扩大你的农场。得益于新的、更直观的界面，员工管理变得更简单了。图层机制将让你能更快看到你的农场中最重要的活动。

在《Farm Manager 2021》中，你将发现更多可以种植的植物种类，其中也包括有机变体！种植欧芹、胡萝卜、黑加仑、醋栗、洋葱等作物，观察它们的生长。全新的机制可以利用可用的员工和机械实现自动农田管理。

在农田劳动时，利用各种各样可以使用的机械。在机械市场购买并出售农业机械。寻找便宜的二手机械。

在《Farm Manager 2021》中，你将发现蜜蜂、绵羊、山羊、鸡、火鸡、鸭、鹅和奶牛。在这里，你将第一次建造渔场，以及鸵鸟、野猪和鹌鹑农场。

物流建筑、机修工车库、兽医诊所等等！这些建筑将帮助你更好地管理你的农场。通过物流中心，你可以更轻松、更快地转运更多资源和产品。在机修工车库，你的农业机械将得到及时修理。来自兽医中心的员工将确保你的农场动物健康状况良好。

在春季准备农田、在夏季照顾植物、在秋季收获农作物，并在冬季在大棚里种植植物。天气可能对你的农场造成危害。暴雨和冰雹可能影响农作物收成，而风暴则可能引起毁坏农场的火灾。

快乐的员工就是好员工！雇佣永久员工和季节员工。让员工保持满意，管理薪酬和加班。通过训练提高他们的技能。请记住，每个人都理应得到休息。不快乐的员工可以随时离开，因此，照顾他们是绝对值得的。
