关于这款游戏

2007年，未来。你是雷克斯柯尔特中士，隶属Mark IV生化突击队。2007年，地球早已被90年代爆发的核战争蹂躏得满目疮痍。东西方之间的冲突已经持续了好几个世代，十年灾难之后的人类依旧在艰难地生存发展。寻找通往和平的新途径势在必行，而美国的生化军队可能已经找到了答案：一件位于遥远岛屿的生物武器。你的任务就是搜集信息然后搞清楚这一切到底是怎么一回事。 游戏特色 超强的生化射手：现在只有一股力量才能阻止这些狂暴的生化杀手，科学怪人，金属鲨鱼，甚至有能从双眼发射激光的凶残嗜血巨龙，而你就是这股力量。你就是雷克斯超利骇柯尔特中士，隶属Mark IV生化突击队，带上你最喜欢的七种高度自定义的强力武器在这个充满怪物的岛屿上大开杀戒吧。你可以用忍者镖，枪械甚至老式炸药等武器来击杀敌人，神挡杀神，佛挡杀佛。 杀坏人，救女主，保世界：你是雷克斯柯尔特中士，一半肉体，一半机械，还有一颗美国心。你要作为Mark IV生化突击队的精英执行任务…一个复仇任务。你要不惜一切代价打倒你的前任指挥官以及他麾下杀人不眨眼的生化杀手们。 经典复古的VHS时代风格：好莱坞动作巨星迈克尔·比恩（出演过《终结者》，《异形》，《深渊》），在这个充满未来生化人和变种人的80年代风格科幻作品中回归了。关掉VCR，接上游戏机，准备好感受《孤岛惊魂3：血龙》带来的超立体全身心的刺激体验吧。

系统需求

MINIMUM:
    • OS *: Windows® XP (SP3) / Windows Vista® (SP2) / Windows 10 (64-bit versions only)
    • Processor: 2.66 GHz Intel® Core™2 Duo E6700 or 3.00 GHz AMD Athlon™ 64 X2 6000+
    • Memory: 2 GB
    • Graphics: 512 MB DirectX® 9.0c–compliant
    • DirectX®: DirectX® 9.0c
    • Hard Drive: 3 GB HD space
    • Sound: DirectX 9.0c–compliant
    • Perpherals Support: Windows-compatible keyboard, mouse, optional controller
    • Requires a UPlay account.
RECOMMENDED:
    • OS *: Windows® XP (SP3) / Windows Vista® (SP2) / Windows 10 (64-bit versions only)
    • Processor: 2.93 GHz Intel® Core™ i3-530 or 3.10 GHz AMD Phenom™ II X2 550 or better
    • Memory: 4 GB
    • Graphics: 1024 MB DirectX 11–complaint or higher
    • DirectX®: DirectX® 11
    • Hard Drive: 3 GB HD space
    • Sound: 5.1 surround sound
    • Perpherals Support: Windows-compatible keyboard, mouse, Xbox 360 Controller for Windows
    • Requires a UPlay account.
* 2024 年 1 月 1 日（PT）起，Steam 客户端将仅支持 Windows 10 及更新版本。
