关于此游戏
在备受赞誉的即时战略游戏《战争黎明》下一个独立扩展包中，指挥六种独特阵营中的任一阵营。你可以建造一支大军，也可以带领一支由精英英雄组成的战斗小队加入战斗，体验为你喜爱的种族专门射击的单人战役。联网游戏，直面第 41 千纪的敌军和快速残酷的战斗。
多种族战役： 史上首次，《战争黎明 II》系列的玩家可以体验六个种族中任意种族的单人战役。
建造大军： 在单人战役过程中，升级英雄，解锁新的单位。
6 个可选种族： 在六个多人种族中任意选择，每个种族都带有独特的超级重装单位。

macOS

最低配置:
操作系统： Windows XP SP2, Windows Vista, Windows 7
处理器: Pentium IV 3.2Ghz, 或者 任意一款dual core 处理器
内存: 1GB (XP), 1.5GB (Vista), 1.5 GB (Windows 7)
硬盘剩余空间: 8.5GB
显卡: 板载 128 MB 显存的显卡并支持 Shader Model 3 (Nvidia GeForce 6600 GT / ATI X1600)
其它要求: STEAM 帐号在线
推荐配置：
操作： Windows XP SP2, Windows Vista, Windows 7
处理器: 所有 Core 2 ,AMD Athlon X2或更高
内存: 2GB 或更高 (XP, Vista, Windows 7)
硬盘剩余空间: 8.5GB
显卡: 板载 256 MB 显存的显卡 支持 Shader Model 3 (Nvidia GeForce 7800GT 或者 任意一款 Nvidia GT200 系列的显卡或更高, ATI X1900 或者 任意一款 HD3600 系列的显卡或更高
其它要求: STEAM 帐号在线
