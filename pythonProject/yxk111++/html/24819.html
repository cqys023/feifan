英雄萨姆3：BFE是一款第一人称动作射击游戏，是黄金时代第一人称射击游戏的光荣回归，回归那个男人就是男人，掩护就是业余，开枪就是爆炸的年代。

作为原创独立游戏和年度最佳游戏-“英雄萨姆：首次遭遇”的前传，英雄萨姆3的故事是在地球最后一次尝试反抗Mental的野兽与雇佣军兵团入侵时发生的。

关键特性

  • 疯狂的动作街机风格 – 扣紧扳机摧毁无尽猛攻的攻击者，或是面对泛滥成灾的Mental蛮兽。没有掩护系统，没有营地喘息-只有你自己对抗他们。对抗他们所有人！
  • 令人恐惧的敌人与生物 – 一个由令人难忘的随从们组成的新战斗营，包括隆隆作响的Scrapjeck和高大的Khnum，以及传奇的无头Kamikaze，Gnaar和Sirian Werebull，它们的残暴组合会让你的战斗充满乐趣！
  • 壮观的环境– 在不远的将来埃及的广大战场上与纯粹的混乱针锋相对。明日破碎的城市和古世界的倒塌神庙是你的毁灭乐园！
  • 毁灭性军火库 -释放英雄萨姆军火库中的武器，包括作战狙击步枪，双筒霰弹枪，自动爆炸霰弹枪，惩戒级转轮机枪和残暴炸裂加农的密集火力！立刻带上萨姆所有的武器，切换枪支，发射最强火力！
  • 猛烈的近战攻击 -狭路相逢勇者胜！撕扯出近身Gnaar的眼睛，扭曲可怕的Scrapjack的脸，或是抓住Arachnoid Hatching的脖子瞬间杀死它！
  • 多人联机合作 -与多达16名玩家在线参加对抗Mental部落的战争，并在标准合作，经典合作，自助合作，野兽狩猎和团队野兽狩猎模式中扫清12个单人关卡中的一切骚乱.
  • 单人与多人生存模式-在生存模式下（离线）对抗特殊关卡中一波又一波的敌人，或者和你线上的朋友加入两组队伍，在残酷的团队生存模式中与敌人互相对抗！
  • 多人对战模式-在不同的对人对战模式中互相对抗或团队对抗，包括死亡竞赛，团队死亡竞赛，夺旗，最后生还模式，最后团队生还模式，瞬间杀戮，我的负担，团队野兽狩猎和团队生存模式！
  • 分屏模式-本地（离线）分屏游玩合作和多人对战模式支持多达四人一起游玩！甚至还支持多键盘和鼠标。
  • 关卡编辑器-创造你自己的关卡，MOD，纹理和其他内容，只需使用全功能英雄编辑器3.5！
  • Steam工坊-提交你的英雄萨姆3自创内容，或者下载新的玩家创造的修改与内容！
  • 其他游戏选项– 5个难度级别,快速保存,自动保存,多个保存字段,第3人称视角,合作模式选项,游戏控制器支持,中文字幕,家长监督和血腥控制(包括嬉皮模式),多样化的色彩主题,大量可供调整的进阶图形选项,秘技,控制面板,Steam卡牌,成就,排行榜和云空间等等!

macOS

最低配置:

    • 操作系统： Windows XP 32-bit (SP2 或者 SP3)
    • 处理器: 英特尔酷睿双核 或 AMD 双核 主频2.0 GHz
    • 内存: 1GB
    • 显卡： nVidia GeForce 7800/7900/8600 系列, ATI/AMD Radeon HD2600/3600 或者 1800/X1900 系列
    • DirectX®等级: 9.0c
    • 硬盘： 4G 剩余空间
    • 声卡: 任何可兼容 DirectX9.0c 的声卡

推荐配置：

    • 操作系统： Windows 7 64位
    • 处理器: 英特尔(i5/i7-系列) 或 AMD (Phenom II系列) 主频 3.0 GHz
    • 内存: 4GB
    • 显卡： nVidia GeForce 480/580 GTX, ATI/AMD Radeon HD 5870/6970
    • DirectX®等级: 9.0c
    • 硬盘： 4GB 剩余空间
    • 声卡: 任何可运行 DirectX9.0c 的声卡
