甚至它可以完成很多事情，但它以使您能够通过光滑的图形界面在各种计算机和控制台上运行经典游戏而广为人知。设置也是统一的，因此配置一劳永逸。

系统需求

    • 最低配置：
    • 操作系统：Microsoft Windows 7
    • 处理器：Intel Pentium 4及更高版本（需要具有SSE2指令的CPU）
    • 内存：32 MB RAM
    • 显卡：任何兼容的OpenGL 2.x或Direct3D11 GPU。为了使着色器正常工作，至少应支持Shader Model 2.0。
    • DirectX版本：11
    • 注意事项：对于OpenGL：Windows 10上一代的Intel HD 2K / 3K GPU必须回退到OpenGL 1.1驱动。
    • 推荐配置：
    • 操作系统：Microsoft Windows 10
    • 处理器：英特尔酷睿系列或与AMD兼容的产品
    • 内存：512 MB RAM
    • 显卡：OpenGL至少需要内核高清4K，Direct3D 11的任何兼容D3D11 GPU至少都应支持Shader Model 3.0和/或4.0。
    • DirectX版本：11
    • 注意事项：对于OpenGL：Windows 10上一代的Intel HD 2K / 3K GPU必须回退到OpenGL 1.1驱动。
