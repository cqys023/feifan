关于这款游戏

《遵命，陛下：寒冬》 是一款具有影视质感的王国管理游戏。 缔结盟约，支持你所爱的人，修复人民对你的信任。 深入这个世界，感受丰富的斯拉夫民俗元素，接见形形色色的角色。 来见之人有的会害你，有的会助你，还有的会让你的王厅想起欢声笑语。 察明他们的意图，决定是否满足他们的诉求。

游戏特色

管理资源 合理分配你的资源，帮助真正需要帮助的人。 派遣属下协助国民，决定将国库的资源分拨给谁。 虽说众口难调，可是你能否坐稳江山，完全取决于农民、领主与国王是配合还是阻碍。 率领部下参战 走上恢弘的战场，左右风云变化的战局。所有人的生死都沉甸甸地落在了你地肩上，不过只要他们愿意效忠国王，就会站在你的左右…… 关心你的家人 日理万机固然辛苦，但你还有家人可以依靠。 即便是压力如山的时候，家也是你可靠的港湾。 两个丫头淘气可爱，逗你开心，新生的孩子也会温暖你的心。 王后永远支持你的大业，即便遇到困难也会勇于面对。 关心你的家人，你们携手，或许就能战胜敌人。 缔结盟约 身为一国之主，结交盟友或许才能稳固疆域，只是各个领主心思各异，很难同时满足他们所有人的要求。帮助他们完成诉求，然后决定应该把谁关进地城。 加入我们的Discord服务器，就能查看常见问题与回答，了解开发的动态与游戏的重要通知啦！

系统需求

最低配置:
    • 操作系统 *: Windows 7 or later
    • 处理器: Intel Core i5
    • 内存: 2 MB RAM
    • 显卡: Nvidia Geforce GTX 550
    • 存储空间: 需要 2 GB 可用空间
    • 声卡: N/A
推荐配置:
    • 操作系统 *: Windows 7 or later
    • 处理器: Intel Core i5
    • 内存: 4 MB RAM
    • 显卡: Nvidia Geforce GTX 550
    • 声卡: N/A
