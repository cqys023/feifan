《战地5》是一款由EA制作并发行的第一人称射击游戏，该作为《战地4》的正统续作。目前分析公司Stern Agee确信本作正在秘密开发中，他们认为EA想要用更多的时间来开发《战地5》，以防止《战地5》重蹈《战地4》的覆辙。

官方表示，游戏将真正达到照片级画质，会加入更多的特性，更多的额外内容，更丰富的摧毁效果，具有新大作的一切元素。

相比之前的1战少了很多有趣好玩的载具，不过通过二战的末日武器将整个游戏的氛围真正的营造出了一种世界大战的感觉，不过服务器的质量并不是特别的稳定加上平衡性的调整，可能对于大部分新玩家而言就不是那么的友好了，不过优质的画面和环境破坏一直是这个游戏的最大亮点，喜欢二战的玩家不要错过了哦。

《战地5》的单人战役对于潜行玩法倚重颇多，并不总能发挥出《战地》系列所长，但这几段战役的确讲述了一些观点尖锐的故事，也在爆炸震天的战斗中呈现了好看又声势浩大的枪战。战役中敌人种类较少、AI也不够一致，这些缺陷都让战役中偏线性的部分显得有些稚拙，好在随后也有更加开放的部分，你可以选出按自己想要的方式达成目标。

最低配置要求：

显卡: NVIDIA GeForce GTX 1050 / GTX 660 2GB

系统: 64-bit Windows 7, Windows 8.1 or Windows 10

处理器 (AMD): AMD FX-8350或(Intel): Core i5 6600K

内存: 8GB RAM

DirectX版本: 兼容11.0版本的显卡或同等

网络: 512 KBPS或更快，稳定的网络连接

硬盘可用空间: 50GB
